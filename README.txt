An analysis is proposed in this paper which help decision makers in Olist company to track sales performance of the company from a historical data during years. 
Concluding this analysis with reports will form a business intelligence suite that will improve the business value.

Key Words: Data Warehousing, Data base, Business Intelligence, Data Analysis, Sales, Olist, Data Science.

Technology and tools used
	Languages used:
		- Structured Query Language (SQL).
		- Multi-Dimensional Expressions (MDX)
	Software used:
		- Microsoft SQL server Management Studio 2019 (for SSIS)
		- Visual Studio 2017 (for SSAS)
		- Tableau (for Dynamic Visualizations)

This Repo Contains: -
	- A report contain all steps made from Data Acquisition to project final output.
	- OLAP Cube.
